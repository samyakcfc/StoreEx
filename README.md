# StoreEx
AI-powered inventory management for small convenience stores.
## What it does
StoreEx helps small store owners stop losing money on expired products. It tracks shelf and backroom stock, monitors expiry dates, and sends plain language alerts to the manager and cashier — in English, Spanish, and Hindi.
## Features

Store Floor Map — color coded shelves (red, orange, green)
Cashier Register — expiry alerts fire when product is selected
AI Stock Monitor — powered by Gemini 2.5 Flash, ranks alerts by urgency
Multilingual — English, Spanish, Hindi

## How to run

Install dependencies: pip install -r requirements.txt
Add your Gemini API key in app.py
Run: python app.py
Open browser: http://127.0.0.1:5000
